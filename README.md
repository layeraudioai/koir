# koir _ mp3toexe
mp3->midi->mml->exe

# building an exe from a mp3 involves
##  mp3 -> midi - make a folder called mp3 with koir/ and place mp3s in there
## midi -> mml  - if you arent starting with mp3s make a folder called midi in koir/ and place midis in there
##  mml ->  c   - if you already have mmls make a folder called mml in koir/ and place mmls in there
##    c -> exe  - if you already have c mmls make a folder called songs in koir/ and place c files in there

# how to: linux
## in terminal goto your koir/ directory
## chmod +x *.sh
## ./compile.sh
## follow the noob proof instructions
